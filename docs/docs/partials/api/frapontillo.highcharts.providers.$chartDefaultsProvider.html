$chartDefaultsProvider
service in module frapontillo.highcharts.providers

Description

$chartDefaultsProvider can be used during the config phase of your app to add new chart defaults options. Those options can later be used by the frapontillo.highcharts.directives.directive:chart directive to retrieve the desired default.

Methods
