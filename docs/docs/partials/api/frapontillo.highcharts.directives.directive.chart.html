chart
directive in module frapontillo.highcharts.directives

Description

Directive to build HighChart charts according to the specified parameters.

Dependencies

  • $filter
  • $log

Usage

as element:
<chart
       [options="{object}"]
       [hidden-series="{Array}"]>
</chart>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
options
(optional)
object

The options configuration object, the same you would pass to HighCharts.

hiddenSeries
(optional)
Array

Array of series indexes to hide in the chart.
